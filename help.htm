Map Tools:

  • All map tools described below are located either on the left side of map. Place mouse over each tool for a brief description.

    • - Zoom In : click tool on left side of map, then click or drag a box around the map area to magnify.
    Zoom In or Out : Additionally, you can zoom in or out pressing the + or - keys on the slider bar. Or move the cursor over a location and use the mouse scroll button to zoom in or out on that location. To center and zoom in on a location, double click the location.
    - Pan or Recenter : hold down mouse key on map and then drag in the direction to move, release mouse button, and map will be moved to new location.
    - Identify or Information : click tool and then click on map point ( or ) to view individual inventory description.
    - Select by Circle : click tool, enter search distance into form below map, and click map (for map location) or Search button (for a specific address). A summary report of the inventories within the circle will appear below the map.
    - Start Over or Zoom to Full Extent : click tool on left side (center) of web page to zoom map out to the starting point and full extent of the data.

Map Symbols:

  • Inventory properties are represented as points on the map.

    • - All Inventory Properties
    All properties in the database. Map number represents the inventory identification number (Scan#).
    - All Selected Inventory Properties
    Currently selected inventory properties. Properties can be queried either with the SEARCH INVENTORY form (left side of map), clicking directly on the map points, or Select by Circle tool.

    Top of Page

Search Inventory:

  • Use the form on left side of map to search for specific inventories. Multiple words or phases may be searched for by separating each with a comma.

    Keywords : Any word or phrase within the entire database. Multiple words or phases may be searched for by separating each with a comma.
    Property Name : Any portion of the historic property name.
    Parcel Number : All or a portion of the 10 digit tax parcel number.
    Years Built : The year a building was built. Multiple years can be searched by separating with commas, for example 1885, 1886, 1887   will return all structures built during the period 1885-1887.
    Architect : Any portion of the architect's name.
    Builder : Any portion of the builder's name.
    Address : Select from a pulldown list any address currently in the database.
    Recorded : Select from a pulldown list the year a property was recorded in the survey inventory.
    Style : Select from a pulldown list any architectural style currently in the database.
    Form/Type : Select from a pulldown list any form/type of building currently in the database.

Other Questions:

  • Some of the features in the map are not always being drawn on the map. Why? Some features may only appear at particular scales to avoid cluttering the map. For instance, the street names appear as you zoom in closer to the map.


  • How can I make the map bigger? Try the F11 key to maximize the browser window. Using F11 again will return the browser window to normal.
