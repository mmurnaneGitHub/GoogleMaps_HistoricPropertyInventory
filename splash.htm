Welcome to the Tacoma Historic Property Database!

This database is designed to be a tool for property owners, investors, researchers and City employees to access the City’s historic properties inventory.

Most of the properties on this database are not listed on any register, but many are eligible.  These properties have been identified by the City during Cultural Resource Surveys between 1978 and 2004 as potentially historically significant.  For information regarding buildings that are already registered, please visit the Historic Preservation Resource Library at www.tacomaculture.org. 

Tips for Use:

  • Turn off any “pop up” blockers on your browser.  Pop-up blockers will interfere with the function of this site.
  • This database can be searched using either map tools, located at the top of the screen, or via text, using the fields on the left.
  • For more successful text searches, try searching in the broadest terms to gain the most results—for instance, if looking for the architecture firm “Heath and Gove”, try searching first with “Heath” alone, and narrow your search from there.
  • This information is intended to provide only a property summary.  Additional information can be gained by visiting the Tacoma Public Library or Tacoma Historical Society.
  • Please note that funding and time constraints limit the number of properties that are inventoried in this database.  Many historically significant properties are not yet recorded!
